# secrete-santa
This site is a challenge from Oracle and Alura to train the logic programming that was learned during the courses and to put it into practice 

## Features:
- Add friends without duplicates.
- Display the list of friends.
- Randomly draw a friend.

## How to Use:
1. Enter a name and click "Add".
2. The names will appear in the list.
3. Click "Draw Friend" to randomly select a name.

## Technologies:
- HTML, CSS, JavaScript.
